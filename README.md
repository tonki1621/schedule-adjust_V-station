**V-station（ボランティアセンター）** の活動を加速させるために開発された、完全カスタムの日程調整・スケジュール管理Webアプリケーションです。

大人数のボランティアスタッフ、プロジェクトメンバー、キャンパスをまたいだ学生たちの複雑なスケジュールを、スマホ・PC問わず直感的な操作で一元管理します。

🌐 **アプリへのアクセス:** [https://schedule-adjust-v-station.streamlit.app/]

---

## 🎯 背景と目的 (Background & Purpose)

ボランティアセンターでは、多数のプロジェクトが同時進行しており、参加する学生のキャンパス（なかもず・もりのみや等）や入学年度、履修している時間割もバラバラです。
従来の日程調整ツールでは「大学の時間割に合わせた調整がしにくい」「回答の集計や絞り込みに時間がかかる」「未回答者へのリマインドが面倒」といった課題がありました。

本アプリはこれらの課題を解決するため、**「V-stationの運用に100%最適化された、超高速・ノーストレスな日程調整体験」**を提供することを目的に開発されました。

---

## ✨ 主要機能 (Key Features)

### 1. 直感的な「なぞり塗り」UI
カスタムJavaScriptで構築された独自のエディタを搭載。PCのマウスドラッグやスマホのスワイプ操作で、「可(◯)」「未定(△)」「不可(×)」をペンで色を塗るように連続入力できます。（※PC版は右クリックで一発消去も可能）

### 2. 3つの柔軟な調整モード
イベントの性質に合わせて最適な調整方法を選択できます。
- **🕒 時間帯モード:** 「4/1〜4/5の 13:00〜18:00（15分刻み）」など、細かいミーティング調整に。
- **🏫 時間割モード:** 月曜〜金曜の「1限〜5限・放課後」の枠を使い、メンバーの空きコマを一括で集計。
- **📅 候補リストモード:** 特定の日時やイベント案をリストアップして多数決を取るアンケート形式。

### 3. V-stationに特化したグループ管理と絞り込み
ユーザーは登録時に自分の属性を設定します。集計時にはこれらの属性でワンクリックでグラフを絞り込むことができます。
- **キャンパス:** なかもず, もりのみや, すぎもと, あべの, りんくう
- **入学年度:** 動的生成（常に最新の年度から過去6年分を自動表示）
- **オプション:** 卒業生ネットワーク関係者 など

### 4. Discord完全連携＆自動メンション
イベントを作成すると、指定したターゲット（例：`@なかもず`, `@2024年度入学生`, `@everyone`）に対して、DiscordのWebhook経由で自動的にメンション付きの通知と回答リンクが送信されます。

### 5. 外部カレンダー(iCal)の自動反映
Googleカレンダー等の「非公開URL(.ics)」をプロフィールに登録しておくと、自分の既存の予定を「授業等(グレー)」として日程調整画面に自動でブロック反映できます。

## 🏗 システム構成 (Architecture)

本アプリは、フロントエンドをStreamlitで構築し、バックエンドのデータベースおよびAPIとしてGoogle Apps Script (GAS) ＋ Google Spreadsheet を採用したサーバーレスアーキテクチャで稼働しています。

- **Frontend:** Streamlit (Python), Custom HTML/JS/CSS Components
- **Backend / Database:** Google Apps Script (REST API化), Google Spreadsheet
- **Hosting:** Streamlit Community Cloud
- **Integration:** Discord Webhook API, Google Calendar (iCal parsing)

## 🚀 使い方 (How to Use)

### 【一般ユーザー向け】
1. **アカウント作成:** 初回アクセス時に氏名、PIN、キャンパス、入学年度を登録します。
2. **時間割の登録:** メニューの「⏰ 時間割設定」から、半期ごとの固定の授業スケジュールを登録しておくと、以降の調整時に「一発反映」が使えて便利です。
3. **回答する:** Discordに届いたリンク、またはアプリトップ画面の「未回答の予定」から対象イベントを開き、自分の予定を塗って提出します。

### 【管理者向け】
1. **イベントの作成:** メニューの「➕ イベント新規作成」から、対象者や期限を設定してイベントを発行します。
2. **集計の確認:** イベント画面の「📊 集計タブ」を開くと、時間帯ごとの参加可能人数がヒートマップ表示されます。キャンパスや年度で絞り込むことも可能です。
3. **未回答者の確認:** 「⚙️ 管理者専用」画面から、特定のイベントの未回答者を抽出し、一覧で確認することができます。
